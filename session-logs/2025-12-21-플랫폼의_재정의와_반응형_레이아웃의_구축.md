# 플랫폼의*재정의와*반응형*레이아웃의*구축

_(부제: 2025-12-21, 이벤트 정보 플랫폼으로의 전환 및 PC/모바일 하이브리드 인터페이스 구현)_

## 발단: 서비스 정체성의 명확화

레오는 기존의 쇼핑몰 형태에서 벗어나 전국의 축제, 전시, 공모전 정보를 제공하는 '이벤트 플랫폼'으로 프로젝트의 방향성을 구체화했다. 그는 디자인적 벤치마킹 대상으로 '펫프렌즈'의 UI 구조를 제시하며, 이를 가보자고 플랫폼의 특성에 맞게 이식할 것을 요청했다. 관찰자인 안티그래비티는 사용자의 의도를 분석하여, 서비스의 메인 색상을 연한 초록색(#4CAF50)으로 확정하고 구조적인 개편에 착수했다.

## 전개: 컴포넌트 단위의 구조 설계

안티그래비티는 브라우저 분석 도구를 활용하여 벤치마킹 대상의 레이아웃 밀도를 측정했다. 이를 기반으로 `SearchBar`, `FloatingBanner`, `FloatingActionButton` 등 핵심 컴포넌트를 독립적인 파일로 분리하여 제작했다. 특히 메인 페이지인 `page.tsx`는 2단 고정 헤더와 가로 스크롤 섹션을 적용하여 정보 탐색의 효율성을 높이는 방향으로 재구성되었다. 모든 시각 요소는 레오가 요청한 초록색 테마를 일관되게 적용하는 방식으로 진행되었다.

## 위기: 디바이스 환경에 따른 레이아웃 불일치

작업 도중 PC 브라우저 환경에서 레이아웃이 겹치거나 부적절하게 배치되는 현상이 발견되었다. 모바일 전용으로 설계된 고정폭 컨테이너와 PC용 브랜딩 영역이 하나의 플렉스 박스 안에서 충돌하면서 발생한 문제였다. 또한, `fixed` 포지션을 사용하는 플로팅 요소들이 모바일 뷰포트 내부가 아닌 전체 화면을 기준으로 정렬되면서 사용자의 시각적 동선에서 벗어나는 기술적 난관이 발생했다.

## 절정: 레이아웃 엔진의 재구성 및 수치 교정

안티그래비티는 `layout.tsx`의 구조를 전면 수정하여 문제를 해결했다. 전체 화면을 좌측 브랜딩 영역과 우측 앱 컨테이너로 명확히 분리하고, 우측 영역에 `lg:shrink-0` 속성을 부여하여 480px의 고정폭을 강제했다. 특히 플로팅 버튼의 위치를 잡기 위해 `lg:right-[calc((100vw-480px)/2+24px)]`와 같은 동적 계산식을 적용했다. 이 수식은 브라우저의 전역 너비와 앱 컨테이너의 너비를 비교하여 항상 컨테이너 우측 하단에 요소가 유지되도록 보정하는 역할을 했다. 레오는 개선된 화면을 확인하며 수정된 레이아웃이 의도에 부합함을 확인했다.

## 결말: 인터페이스의 안정화와 정체성 확립

이번 세션을 통해 가보자고 플랫폼은 단순한 클론 코딩을 넘어 독자적인 브랜딩과 PC/모바일을 아우르는 안정적인 인터페이스를 확보했다. 특히 PC에서 모바일 환경을 시뮬레이션하는 'App-like Web' 레이아웃을 성공적으로 구현함으로써 사용자 경험의 일관성을 제공할 수 있게 되었다. 레오와 안티그래비티는 이 과정을 통해 시각적 디자인뿐만 아니라 수학적 배치의 중요성을 확인하며 프로젝트의 기술적 완성도를 한 단계 높였다.

### 🛠 기술적 각주 (Technical Summary)

- **수정된 파일 (Modified Files)**:
  - `src/app/layout.tsx`: PC/모바일 이원화 레이아웃 로직 구현
  - `src/app/page.tsx`: 이벤트 중심의 멀티 섹션 구조 적용
  - `src/components/SearchBar.tsx`: 테마 중심의 검색 컴포넌트 생성
  - `src/components/FloatingBanner.tsx`: 컨디셔널 렌더링 및 위치 최적화
  - `src/components/FloatingActionButton.tsx`: 뷰포트 기반 동적 위치 계산 적용
- **주요 기술 (Key Tech Stack)**:
  - **Tailwind CSS**: 반응형 유틸리티 클래스와 동적 수치 계산식(calc) 활용
  - **Next.js App Router**: 레이아웃 상속 구조를 통한 전역 UI 상태 관리
- **핵심 기능 (Key Features)**:
  - **Dual-View Layout**: PC 환경에서 전용 브랜딩 영역과 모바일 뷰를 동시 렌더링
  - **Green Theme Branding**: 일관성 있는 색상 토큰을 활용한 브랜드 아이덴티티 구축
  - **Sticky Interaction**: 스크롤 깊이에 따른 다중 요소 고정 기능 구현
