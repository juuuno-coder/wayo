# 루비의 침묵과 달력의 태동

_(부제: 2025-12-21, 백엔드 의존성 제거 및 사용자 여정 달력 구현)_

## 발단: 현실적인 우회

레오는 백엔드 환경인 루비(Ruby on Rails)를 실행하지 않고도 개발을 지속할 수 있는지에 대해 의문을 제기했다. 로컬 환경의 제약으로 인해 백엔드 서버를 상시 가동하기 어려운 상황이었고, 이는 프론트엔드 UI/UX 구현에 병목 현상을 일으키고 있었다. 안티그래비티는 이에 대해 긍정하며, 실제 API 호출 대신 가짜 데이터(Mock Data)를 활용하여 개발 흐름을 유지할 것을 제안했다.

## 전개: 데이터의 모사

그들은 검색 페이지와 찜 목록, 그리고 사용자 프로필 페이지를 우선적인 작업 대상으로 선정했다. 안티그래비티는 `search/page.tsx`와 `likes/page.tsx`에 내장된 페치(fetch) 로직을 제거하고, 약 0.5초의 지연 시간을 포함한 가짜 비동기 함수로 대체했다. 또한, 개발 편의를 위해 `profile/page.tsx`에 '개발용 바로 로그인' 버튼을 삽입하여 복잡한 인증 절차 없이도 서비스의 내부 진입이 가능하도록 설계를 변경했다.

## 위기: 기억의 조각

작업이 진행되던 중 레오는 과거에 언급했던 '캘린더' 기능의 진행 상황을 확인했다. 기존의 여권 페이지(`passport/page.tsx`)에는 방문 기록이 단순한 리스트 형태로 나열되어 있어, 사용자가 자신의 여정을 시각적으로 파악하기에는 한계가 있었다. 특히 찜한 일정과 실제 방문한 일정을 하나의 타임라인에서 관리하고 싶어 했던 레오의 구상을 구체화할 필요가 있었다.

## 절정: 시간의 시각화

안티그래비티는 기존의 단순 리스트 뷰를 걷어내고 전면적인 월간 달력 UI 구현에 착수했다. `CalendarSection` 컴포넌트를 설계하여 날짜별 그리드를 배치하고, 방문 인증이 완료된 날에는 빨간색 점(🔴), 가고 싶은 일정을 찜한 날에는 초록색 점(🟢)이 표시되도록 로직을 짰다. 날짜를 클릭하면 하단에 그날의 상세 정보 카드가 나타나는 인터랙션을 추가했다. 구현 과정에서 발생한 타입 불일치와 태그 누락 문제는 전체 코드 재작성을 통해 해결되었고, 최종적으로는 여권의 감성과 일정 관리의 실용성이 결합된 인터페이스가 완성되었다.

## 결말: 정리와 회고

레오는 완성된 캘린더와 우회된 시스템을 확인한 뒤 작업을 마무리하기로 결정했다. 이번 세션을 통해 프론트엔드 개발이 반드시 백엔드 시스템과 동기화되어야 할 필요는 없음을 확인했다. 데이터의 모사를 통해 구현 속도를 높였으며, 결과적으로 '가보자고' 프로젝트는 단순한 상품 나열을 넘어 사용자의 시간을 관리하는 플랫폼으로서의 정체성을 한 단계 더 다지게 되었다.

### 🛠 기술적 각주 (Technical Summary)

- **수정된 파일 (Modified Files)**:
  - `frontend/src/app/search/page.tsx`: 백엔드 검색 API를 Mock 데이터 필터링 로직으로 대체.
  - `frontend/src/app/likes/page.tsx`: 인증 체크 우회 및 Mock 상품 데이터 적용.
  - `frontend/src/app/profile/page.tsx`: 메뉴 구성 업데이트 및 개발 전용 퀵 로그인 기능 추가.
  - `frontend/src/app/passport/page.tsx`: 월간 달력 UI 개발 및 방문/찜 마커 시스템 구축.
- **주요 기술 (Key Tech Stack)**:
  - **React Context & LocalStorage**: 브라우저 스토리지를 활용한 임시 인증 및 상태 유지.
  - **JavaScript Date Objects**: 복잡한 날짜 연산을 통해 월간 그리드 및 윤달 이슈 대응.
- **핵심 기능 (Key Features)**:
  - **Activity Calendar**: 방문 기록과 관심 일정을 서로 다른 색상의 점으로 시각화한 대시보드.
  - **Mock-driven Development**: 백엔드 서버 부재 상황에서도 완전한 UI 시나리오 테스트 지원.
