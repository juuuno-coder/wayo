## 📱 페이지 구조

### 메인 페이지

1. `/` - 홈 (상품 피드)
2. `/items/[id]` - 상품 상세
3. `/search` - 검색
4. `/cart` - 장바구니
5. `/likes` - 찜한 상품
6. `/orders` - 주문 내역
7. `/orders/[id]` - 주문 상세
8. `/profile` - 마이페이지
9. `/login` - 로그인
10. `/signup` - 회원가입

### 하단 탭바 (5개)

- 🏠 홈
- 🔍 검색
- 🛒 장바구니
- ❤️ 찜
- 👤 My

---

## 🗄️ 데이터베이스 구조

### 테이블 목록

1. **users** - 회원 정보
2. **items** - 상품 정보
3. **likes** - 찜하기 (User-Item)
4. **cart_items** - 장바구니 (User-Item + 수량)
5. **orders** - 주문 정보
6. **order_items** - 주문 상품 (Order-Item)
7. **reviews** - 리뷰 (User-Item + 별점/내용)

## 🧪 테스트 시나리오

### 1. 회원가입 & 로그인

1. `/signup`에서 이메일/비밀번호 입력
2. 자동으로 로그인 페이지로 이동
3. 로그인 후 메인 페이지로 이동

### 2. 상품 둘러보기

1. 메인 피드에서 상품 클릭
2. 상세 페이지에서 이미지, 설명, 리뷰 확인
3. 하트 버튼으로 찜하기
4. 장바구니 담기

### 3. 검색

1. 하단 탭바 "검색" 클릭
2. 키워드 입력 (예: "간식")
3. 검색 결과 확인

### 4. 장바구니 & 주문

1. 하단 탭바 "장바구니" 클릭
2. 수량 조절 (+/-)
3. 주문하기 버튼 클릭
4. 배송지 입력
5. 주문 완료

### 5. 주문 내역 확인

1. 마이페이지 진입
2. "주문 내역" 클릭
3. 주문 상세 확인

### 6. 리뷰 작성

1. 상품 상세 페이지 진입
2. "리뷰 작성" 버튼 클릭
3. 별점(1-5) 입력
4. 리뷰 내용 입력 (최소 10자)
5. 등록 완료

## 📱 앱 스토어 출시 준비

### 1. Capacitor 설치

```bash
cd frontend
npm install @capacitor/core @capacitor/cli
npm install @capacitor/android @capacitor/ios
```

### 2. 설정

```bash
npx cap init
# App name: Gabojago
# App ID: com.gabojago.app
```

### 3. Next.js Static Export

`next.config.ts`:

```typescript
const nextConfig = {
  output: "export",
  images: { unoptimized: true },
};
```

### 4. 빌드 & 동기화

```bash
npm run build
npx cap add android
npx cap add ios
npx cap sync
```

### 5. 앱 아이콘 & 스플래시

- 아이콘: 1024x1024 PNG
- 스플래시: 2732x2732 PNG

### 6. 스토어 등록

- **Google Play**: $25 (일회성)
- **Apple App Store**: $99/년

## 📝 개발 노트

### 한글 경로 문제 해결

- Windows 환경에서 한글 사용자명으로 인한 경로 문제 발생
- `bin/dev.ps1` 스크립트로 UTF-8 인코딩 자동 설정
- Bootsnap 비활성화로 캐시 충돌 방지

### 서버 재시작 최소화

- Rails의 Hot Reload 기능 활용
- 컨트롤러, 모델 수정 시 서버 재시작 불필요
- 초기화 파일(initializers) 수정 시에만 재시작 필요
